# BFHL Node Analyzer

This is my submission for the SRM Full Stack Engineering Challenge.

## Details
- Roll Number: RA2311003010626

## Overview
- **Frontend**: Vanilla JS, raw CSS, HTML5. Uses a custom parsing algorithm to format the output as ASCII trees for the editorial UI design.
- **Backend**: Node.js with Express. Implements custom graph data structures, mapping, and a cycle-detection DFS algorithm to parse the node hierarchies as requested.

## To Run
1. Backend: `cd backend && npm install && node index.js` (Runs on port 3000)
2. Frontend: Open `frontend/index.html` either locally or using Live Server. Ensure `app.js` is pointed to the correct backend host.
